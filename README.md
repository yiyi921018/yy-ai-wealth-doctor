# Y&Y AI Wealth Doctor

AI-powered asset allocation and financial health diagnostic platform for financial advisors.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- TailwindCSS
- shadcn-style UI components
- Recharts
- Framer Motion
- Zustand

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Product Flow

1. Welcome
2. Basic Information
3. Asset Selection
4. Asset Value Input
5. Results Dashboard

## Analysis Engine

The app calculates actual allocation, AI-recommended allocation by age group, asset gap analysis, liquidity/growth/defense scores, overall health score, dynamic insights, and top three improvement recommendations.
